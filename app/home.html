Welcome to the Sonatype UI Gallery

The NX Style Gallery

The NX style gallery is where you can find reusable styles (and hopefully soon components) for use in new or existing Sonatype applications. The styles described here are based off the current IQ styling. You can find descriptions of the various styles, including measurements here: https://docs.sonatype.com/display/INSIGHT/UI+Guidelines The purpose of this site is to take those requirements and turn them into HTML/SCSS that can be used in any project.

Assets and dependencies

Individual _example.scss files will be linked on the example pages. The compiled SCSS for all of the examples can be found here: http://0.0.0.0:4041/assets/nx-styles.css We use Font Awesome ( https://fontawesome.com ) for icons, and in many cases styles are built in to support Font Awesome styles.

How to add a new component to the gallery

* create an html page using the following structure (see template.html) ```html <item-description> Usage docs <marked> You can use markdown inside <marked> tag </marked> </item-description> <item-example-html title="My Example"> example html </item-example-html> ``` * map the component name to the html page created above (add an entry to `config.js`) ```js 'component-name' : 'components/component-name.html', ``` * (only for js components) add necessary imports and angular module dependencies in `config.js`
