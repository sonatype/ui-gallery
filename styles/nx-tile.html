`.nx-tile` is the base building block of our pages. There are three default classes that can be used within an `.nx-tile` `.nx-tile-header` is used for tile titles, it has title and sub-title elements (optional) `.nx-tile-content` for the tile content `.nx-tile-footer` for footer contents (buttons for example) `.nx-tile` can also be paired with `.nx-alert` styles to create tiles with alert coloring.

Required files
