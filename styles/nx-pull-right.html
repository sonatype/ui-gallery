**.nx-pull-right** is a simple helper class that floats an object to the right. Note that if the object is full width you are unlikely to see any change. Note that you might need to [clear the floats](https://css-tricks.com/all-about-floats/).

Required files:

  • _nx-globals.scss
This <div> has a width of 50%. Extra styles added inline to make example more clear.
